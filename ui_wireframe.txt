NAVIGATION FLOW
---------------

Home Page (Landing)
    │
    ├─> Upload Resume Page
    │   └─> Drag & Drop / Browse
    │       └─> Processing Pipeline
    │           └─> Success/Error Message
    │               └─> Return to Home or Upload Another
    │
    └─> Search Candidates Page
        └─> Enter Job Description
            └─> Auto-extract Requirements
                └─> Search
                    └─> View Results
                        └─> View Profile / Download / Contact


RESPONSIVE BREAKPOINTS
----------------------

Desktop: 1200px+ (Full layout as shown)
Tablet:  768px-1199px (Stack components vertically, maintain functionality)
Mobile:  <768px (Single column, simplified drag-drop, touch-friendly buttons)

KEY FEATURES
------------

1. Resume Upload Page:
   - Drag & drop functionality with visual feedback
   - File type validation (PDF, DOCX)
   - Real-time processing status updates
   - Recent upload history with status
   - Batch upload capability (future enhancement)

2. Job Description Search Page:
   - Large text area for JD input
   - Auto-extraction of key requirements using NLP
   - Editable requirements before search
   - Ranked candidate results with match scores
   - Detailed candidate cards with key information
   - Quick actions (view, download, contact)
   - Pagination for results

3. Common Features:
   - Responsive navigation bar
   - Loading states with spinners
   - Error handling with user-friendly messages
   - Session management for recruiter accounts (future)
   - Export functionality for search results (future)
